# Skill_Browser
## Function Description
![Feature preview](https://gitee.com/xiao_yong_Zhang/image-bed/raw/master/2026/%E6%8A%80%E8%83%BD%E6%B5%8F%E8%A7%88%E5%99%A8.png)

The local skills browser allows switching between different directories to search and display local skill lists. It supports automatic skill classification and grouping, and enables quick copying of skill names via a dedicated copy button.
